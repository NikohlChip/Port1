Definition and Usage

The <link> tag defines the relationship between the current document and an external resource. This tag is most often used to link to external style sheets. This element is an empty element, it contains attributes only.

Browser Support

  • Chrome
  • Edge
  • Firefox
  • Safari
  • Opera
